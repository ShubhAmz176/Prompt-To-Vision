# Prompt-To-Vision

## Overview
**Prompt-To-Vision** is an AI-powered text-to-image generation project that converts user prompts into visual images. The system uses **Stable Diffusion** for generating images and integrates models from **Hugging Face** with a **Python backend**.

This project demonstrates how modern generative AI models can transform natural language prompts into creative visual outputs.

---

## Features
- Generate images from text prompts
- Uses **Stable Diffusion** for high-quality image generation
- Integrated with **Hugging Face** models
- Simple **Python backend** for handling requests
- Easy to run locally

---

## Tech Stack
- **Backend:** Python  
- **AI Model:** Stable Diffusion  
- **Model Platform:** Hugging Face  
- **Libraries:** Diffusers, Transformers, Torch  

---

## How It Works
1. User enters a text prompt.
2. The prompt is sent to the backend.
3. The backend uses Stable Diffusion from Hugging Face.
4. The model generates an image based on the prompt.
5. The generated image is returned to the user.

---

## Commands to Run

Run the following commands in the terminal:

```
cd BACKEND
python app.py
```

---

## Example Prompt

```
"A futuristic cyberpunk city at night with neon lights"
```

The system will generate an image corresponding to the given prompt.
